# react-mobx-starter
进行 React + Mobx + React Route 开发的脚手架

## 安装rich-tao-cli脚手架
``` bash
npm install [-g] rich-tao-cli
```


## 下载项目
``` bash
    rich-tao-cli init tpl-a yourProjectName
```



## 运行项目
``` bash
npm run start
```

打开浏览器，输入：`localhost:8000` 查看项目运行效果



## 项目打包
``` bash
npm run build
```
